Background

Use web image

  1. Download and install Firefox from the App Center
  2. Navigate to the picture using Firefox.
  3. Right click the image and select Set As Desktop Background...
  4. Click Set Desktop Background.

Use web image

  1. Download and install Firefox from the App Center
  2. Navigate to a local image.
  3. Right click the image and select Open with> Firefox.
  4. In Firefox, right click the image and select Set As Desktop Background...
  5. Click Set Desktop Background.
